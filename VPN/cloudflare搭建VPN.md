# Cloudflare 域名托管与 Pages 部署教程

本文介绍如何注册免费域名、托管到 Cloudflare，并创建 Cloudflare Pages 项目。

---

# 一、注册免费域名

## 1. 注册账号

打开网站：

```text
https://my.dnshe.com
```

注册并登录账号。

---

## 2. 注册免费域名

登录后：

1. 点击左侧 **免费域名**；
2. 点击 **立即注册**；
3. 选择根域名，建议：

```text
ccwu.cc
```

该域名支持约 **10 年免费使用**。

4. 输入自己喜欢的前缀，例如：

```text
lpw
```

最终域名示例：

```text
lpw.ccwu.cc
```

点击 **确认注册**。

如果提示已经存在，则修改前缀重新注册即可。

注册成功后，可以看到剩余使用时间，大约为：

```text
3653 天
```

到期后可继续免费续期。

---

# 二、注册 Cloudflare

打开官网：

```text
https://www.cloudflare.com
```

按照提示完成注册即可。

---

# 三、托管域名到 Cloudflare

## 1. 添加域名

登录 Cloudflare 后：

1. 搜索 **域名**
2. 点击 **添加域名**
3. 选择 **连接现有域名**
4. 输入刚刚注册的免费域名
5. 点击 **继续**

---

## 2. 选择套餐

选择：

```text
Free（免费计划）
```

继续下一步。

---

## 3. DNS 检查

Cloudflare 会提示：

> 没有检测到 DNS 记录

这是正常现象。

直接点击：

```text
继续前往激活
```

即可。

---

## 4. 修改 DNS 服务器

Cloudflare 会提供两个 Name Server，例如：

```text
xxxxx.ns.cloudflare.com
yyyyy.ns.cloudflare.com
```

复制这两个地址。

然后回到免费域名管理后台：

进入：

```text
DNS 服务器
```

分别填入：

第一行：

```text
xxxxx.ns.cloudflare.com
```

第二行：

```text
yyyyy.ns.cloudflare.com
```

保存设置。

---

## 5. 等待生效

返回 Cloudflare。

点击：

```text
我已更新名称服务器
```

稍等几分钟。

期间可以点击：

```text
立即检查域名
```

或者刷新页面。

当看到：

```text
您的域现在受 Cloudflare 保护
```

说明域名托管成功。

---

# 四、创建 Workers KV

进入 Cloudflare 控制台。

搜索：

```text
Workers KV
```

点击进入。

点击右上角：

```text
创建
```

输入空间名称，例如：

```text
luopw_kv
```

点击创建即可。

---

# 五、创建 Cloudflare Pages

进入：

```text
计算
```

选择：

```text
Workers 和 Pages
```

点击：

```text
创建应用程序
```

不要选择 Worker。

选择：

```text
Pages
```

然后：

```text
拖放文件部署
```

输入项目名称，例如：

```text
luopw
```

点击创建。

---

# 六、上传项目

下载准备好的项目压缩包。

返回 Cloudflare。

点击：

```text
上传内容
```

选择压缩包。

上传完成后点击：

```text
部署站点
```

等待部署完成。

随后点击：

```text
继续处理项目
```

---

# 七、配置环境变量

进入：

```text
设置
```

找到：

```text
变量和密钥
```

点击：

```text
添加变量
```

例如：

变量名称：

```text
ADMIN
```

变量值：

```text
admin123
```

> **注意：**
>
> 变量名称必须使用大写。

保存即可。

---

# 八、绑定 KV 命名空间

进入：

```text
绑定
```

点击：

```text
添加绑定
```

选择：

```text
KV 命名空间
```

填写：

变量名称：

```text
KV
```

KV 空间：

```text
luopw_kv
```

点击保存。

> **注意：**
>
> 变量名称必须为：
>
> ```text
> KV
> ```

---

# 九、绑定自定义域名

进入：

```text
自定义域
```

点击：

```text
设置自定义域
```

输入之前注册好的域名。

继续下一步。

按照 Cloudflare 页面提示，在域名 DNS 中添加对应的 **CNAME** 记录。

等待 DNS 生效。

当域名状态显示：

```text
活动（Active）
```

说明绑定成功。

---

# 十、重新部署

完成域名绑定后：

1. 点击 **创建部署**
2. 再次上传项目文件
3. 点击 **保存并部署**
4. 等待部署完成

部署成功后，即可通过绑定的域名访问项目。

---

# 常见问题

## 域名一直没有生效

建议：

* 检查 Name Server 是否填写正确；
* 等待 DNS 全球同步（通常几分钟到数小时）；
* 在 Cloudflare 点击 **立即检查域名**。

---

## Pages 部署失败

建议检查：

* 上传文件是否完整；
* 项目结构是否正确；
* 是否成功绑定 KV；
* 环境变量是否配置完成。

---

## KV 无法读取

重点检查：

* 变量名称是否为：

```text
KV
```

* 是否绑定到了正确的 KV 命名空间。

---

# 总结

整个部署流程如下：

1. 注册免费域名；
2. 注册 Cloudflare；
3. 域名托管到 Cloudflare；
4. 创建 Workers KV；
5. 创建 Pages；
6. 上传项目；
7. 配置环境变量；
8. 绑定 KV；
9. 绑定自定义域名；
10. 完成部署并访问站点。
