# Cloudflare 搭建免费 VPN 教程

本文介绍如何利用 Cloudflare 的免费服务，结合免费域名和 Workers/Pages，搭建属于自己的免费 VPN 节点，并支持 V2Ray 自动订阅。

---

# 一、注册免费域名

打开 DNSHE 免费域名网站：

```
https://my.dnshe.com
```

操作步骤：

1. 注册并登录账号；
2. 在左侧菜单栏点击 **免费域名**；
3. 点击 **立即注册**（有五个免费额度）；
4. 选择根域名（推荐 `ccwu.cc`，有十年免费时间）；
5. 输入前缀（如 `lpw`），点击 **确认注册**；

> **注意：**
>
> 如果提示已注册过，更换前缀重新注册即可。

注册成功后可以看到使用天数（3653 天），到期后仍可免费续期。

---

# 二、注册 Cloudflare

打开 Cloudflare 官网：

```
https://www.cloudflare.com
```

按照要求注册账号即可。

---

# 三、托管域名

将第一步注册的免费域名托管到 Cloudflare。

## 3.1 添加域名

1. 登录 Cloudflare，在左侧搜索栏搜索 **域名**；
2. 进入后选择 **添加域名** → **连接域名**；
3. 填写刚刚注册的域名，点击 **继续**；
4. 选择 **Free**（免费）计划；
5. 在最下方点击 **继续前往激活**；
6. 提示无 DNS 记录，直接点击 **确认**。

## 3.2 配置 DNS 服务器

1. 步骤二会提供两个 Cloudflare DNS 服务器地址，**复制第一个**；
2. 回到 DNSHE 免费域名管理，选择 **DNS 服务器**；
3. 将第一个 DNS 服务器粘贴到第一行；
4. 回到 Cloudflare 复制第二个 DNS 服务器，粘贴到第二行；
5. 点击 **保存设置**；
6. 返回 Cloudflare，点击 **我已更新名称服务器**。

## 3.3 等待生效

Cloudflare 会提示：

```
正在等待注册机构传播新的名称服务器
```

一般等待几分钟即可。点击 **立即检查域名**，如果尚未生效按 **F5** 刷新页面，直到出现：

```
您的域现在受 Cloudflare 保护
```

说明域名托管成功。

---

# 四、创建 KV 空间

1. 在 Cloudflare 左侧搜索栏搜索 **KV**；
2. 进入 **Workers KV**；
3. 点击右上角 **创建**；
4. 输入空间名称（如 `luopw_kv`），点击 **创建**。

---

# 五、创建 Pages 项目

## 5.1 创建应用

1. 在 Cloudflare 左侧找到 **计算** → **Workers 和 Pages**；
2. 点击 **创建应用程序**；
3. 下方选择 **Pages** → **开始使用**；
4. 选择 **拖放文件** → **开始使用**；
5. 填写项目名称（如 `luopw`），点击 **创建项目**。

## 5.2 上传部署文件

在新的浏览器窗口打开下方地址，下载 Pages 专属安装包：

```
https://github.com/cmliu/edgetunnel/archive/refs/heads/main.zip
```

下载完成后：

1. 返回 Cloudflare，点击 **上传内容**，将压缩包上传；
2. 点击 **部署站点**，等待部署完成；
3. 部署完成后，点击 **继续处理项目**。

## 5.3 配置环境变量

在项目菜单栏中找到 **设置**：

1. 在 **构建** 中找到 **变量和密钥**，点击 **添加**；

| 变量名称 | 值 | 说明 |
| - | - | - |
| `ADMIN` | `admin123` | 用户名和密码，变量名必须大写 |

2. 点击 **保存**。

## 5.4 绑定 KV 空间

在 **绑定** 中点击 **添加**：

1. 选择 **KV 命名空间**；

| 变量名称 | KV 空间名称 | 说明 |
| - | - | - |
| `KV` | `luopw_kv` | 变量名必须大写 |

2. 点击 **保存**。

## 5.5 绑定自定义域名

1. 在项目菜单栏中点击 **自定义域名**；
2. 选择 **设置自定义域名**；
3. 填写免费域名，点击 **继续**；

> **提示：**
>
> 在左上角 Cloudflare 图标处右键点击，选择 **在新的标签页中打开链接**，避免关闭当前页面。

## 5.6 配置 DNS 记录

1. 切换到新标签页；
2. 点击刚刚绑定的域名；
3. 右侧找到 **DNS 记录** → **添加记录**；

| 类型 | 说明 |
| - | - |
| `CNAME` | |

4. 回到第一个 Cloudflare 页面，复制 **名称**；
5. 回到第二个页面，粘贴到 **名称**；
6. 回到第一个页面，复制 **内容**；
7. 回到第二个页面，粘贴到 **内容**；
8. 点击 **保存**。

## 5.7 激活域名

1. 返回第一个 Cloudflare 页面，看到新记录后点击 **激活域**；
2. 等待状态变为 **活动**，说明激活成功。

## 5.8 重新部署

1. 点击 **创建部署**；
2. 选择 **上传内容**，将安装包重新上传一次；
3. 点击 **保存并部署**；
4. 点击 **继续处理项目**，等待自动跳转。

---

# 六、配置后台

1. 点击部署的域名，进入部署页面；
2. 在浏览器地址后添加 `/admin` 进入后台；
3. 输入之前配置的密码（如 `admin123`），点击 **登录**；
4. 在 **优先订阅生成** 中按需配置；

> **推荐配置：**
>
> 端口建议选择 **443**（加密端口）。

5. 点击 **保存**。

---

# 七、配置 V2Ray 订阅

## 7.1 添加订阅分组

1. 打开 V2Ray，选择 **订阅分组**；
2. 进入 **订阅分组设置**；
3. 点击 **添加**；
4. **别名** 自定义填写；
5. **可选地址** 粘贴刚才的自适应订阅链接；
6. 点击 **确定**。

## 7.2 更新订阅

1. 切换到刚刚创建的分组；
2. 选择 **订阅分组** → **更新全部订阅（不通过代理）**；
3. 稍等片刻即可获得免费节点。

# 常见问题

## 域名托管失败

* 确认 DNS 服务器地址已正确粘贴；
* 等待 DNS 传播（通常需要几分钟）；
* 按 F5 刷新页面重试。

## Pages 部署失败

* 确认上传的压缩包为 edgetunnel 官方安装包；
* 检查 KV 绑定是否配置正确；
* 确认环境变量名称均为大写。

## 无法访问后台

* 确认域名 DNS 记录已配置正确；
* 确认域名状态为 **活动**；
* 确认在 URL 后添加了 `/admin`。

# 总结

整个搭建流程如下：

1. 注册免费域名（DNSHE）
2. 注册 Cloudflare 账号
3. 托管域名到 Cloudflare
4. 创建 KV 空间
5. 创建 Pages 项目并部署
6. 绑定域名并配置 DNS
7. 配置后台管理
8. 配置 V2Ray 订阅

完成以上步骤后，即可免费使用 Cloudflare 搭建的 VPN 节点，并通过 V2Ray 自动更新订阅。
